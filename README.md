# Vehicle Showroom Management System

A full stack web application for managing a vehicle showroom. The system helps manage vehicles, customers, bookings, and sales in one place. It provides separate access for users and administrators, making it easy to handle daily showroom operations through a simple and clean interface.

## Features

* Secure user login
* Role based access
* Vehicle management
* Customer management
* Vehicle booking
* Sales management
* Search available vehicles
* Input validation
